Getting Started

UI Module

Create an ui module like portal-sample-module.

Take a look at the UI components library

This ui module can be tested via maven profile -Pwildfly in a mock deployment in wildfly controlled by you IDE (Eclipse or IntelliJ).

Assembly

Create an assembly like cui-portal-module-sample-assembly that contains the ui module.

This assembly should contain a CUI portal authentication module, e.g. portal-authentication-servlet-container.

As default a tanuki based stand alone application including latest tomcat will be created. To create a docker image just call "mvn clean install -P docker".
